## Notes:

- **GST Format:** Indian GST numbers are 15 characters (2 state code + 10 PAN + 1 check digit + 1 Z + 1 check digit)
- **PAN Format:** Indian PAN is 10 characters (5 letters + 4 digits + 1 letter)
- All contact numbers are 10-digit Indian mobile numbers
- Email addresses follow standard format
- Addresses include complete Indian addresses with PIN codes

## Testing Tips:

1. **Test Validation:** Try submitting with empty fields to test required field validation
2. **Test Duplicate GST/PAN:** Try adding the same GST or PAN twice to test uniqueness
3. **Test Email Format:** Try invalid email formats to test validation
4. **Test Special Characters:** Use names with special characters to test input handling
5. **Test Long Text:** Use very long company names/addresses to test UI responsiveness
